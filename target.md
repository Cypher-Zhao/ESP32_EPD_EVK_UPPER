# ESP32 EPD EVK UPPER

## 1. 硬件

下位机是ESP32S3，可以通过串口或者tcp通讯
此上位机是以网页实现的，网页上通过串口或者tcp通讯控制下位机并传输数据，ip地址为局域网内地址，不考虑外网使用

## 2. 功能

首先实现 tcp通讯，能够输入目标地址和端口，然后连接成功后，可以发送消息给下位机（发送功能测试用，实际上并不通过编辑消息的方式发送）
